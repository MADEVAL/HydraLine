# hydraline_server

Pure-Dart server for [Hydraline](../../README.md): shelf / Dart Frog middleware,
single-pass in-order SSR streaming, HTMX helpers, caching and bot-aware
delivery with a byte-identical document body (no cloaking, I3).

No `package:flutter` (invariant I1).

> Status: pre-MVP skeleton (Phase 0). Server surface lands in Phase 2 — see
> `specs/PHASE_2_PLAN.md`.
